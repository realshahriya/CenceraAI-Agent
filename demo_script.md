**Goal**: Show the judges that the agent is "Alive" on both Telegram and the Web, and that memory is shared.

---

## Scene 1: Introduction (Camera: Web Dashboard)

**Action**: Start recording with `http://localhost:3000` open. Wallet is NOT connected yet.

## Scene 3: Cross-Platform Interaction (Telegram)

**Action**: Switch to Telegram. Clear history if needed.

**Action**: Switch BACK to the Web Dashboard (`localhost:3000`).
**Action**: Type in the web chat: `What is the secret code I just told you on Telegram?`
**Voiceover**: "Now, I'm on the web interface. I haven't told the web app anything. Let's ask it."
**Expected Response**: The AI replies with "BNB-Winner-2024".
**Voiceover**: "And there it is. The memory is decentralized and shared instantly across platforms."

## Scene 5: Autonomous Evolution (Backend Console)

**Action**: Show the VS Code Terminal (Backend logs).

**Voiceover**: "Finally, this agent is alive. Even when I'm not talking to it, this background process runs every minute to evolve its personality and update its on-chain stats."
